# Finance Control System

Sistema completo de controle financeiro pessoal desenvolvido com:

- Java
- Spring Boot
- PostgreSQL
- React
- Docker

Projeto construído como parte de um estudo completo de engenharia de software.

## Arquitetura

Backend:
Java + Spring Boot

Frontend:
React

Banco:
PostgreSQL

Infraestrutura:
Docker

CI/CD:
GitHub Actions
